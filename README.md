# fdf
42 fdf-project
FDF or "fils de fer" of wireframe.
c program that takes in a map of height data and displays it on a window as wireframe.
Compiling requires [minilibx](https://github.com/42Paris/minilibx-linux) and uses make.
Usage: ./fdf <map>

- p: perspective mode
- o: orthographic mode
- arrows: move
